(Unity VR) Generative Music App

A Unity-based prototype which can generate ambient soundscapes out of given audio files.

The currently built scene spawns spheres around the listener at randomized time intervals. Each sphere plays a random audio loop (recorded by me) out of a given selection. The spheres go on to move around the listener, resulting in a constantly changing soundtrack.

The project can be found on my GitHub page.
