(.NET) Generative Art App

A .NET/C# console app which generates abstract art based on numeric input from the user.

The current version of the app has images linked to each number in the 0-9 range. It analyzes user input and calls up one of those images for each number found in it, randomizes each image's position and rotation, combines the images, and sends them through randomized post-processing.

I'm planning to extend the app's functionality by adding more images for letters and other symbols (including multiple images per symbol), as well as to turn it into a web app with Blazor.

The project can be found on my GitHub page.
