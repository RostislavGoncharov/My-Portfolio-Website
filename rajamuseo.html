Case Rajamuseo

A Unity-based educational app for Rajamuseo (the Border Guard Museum) in Imatra, Finland.

My contributions to the project include:

- programming multiple UI interactions (screen transitions, input fields, sliders, etc.)

- programming a Scriptable Object-based language selection system

- recording a few sounds

- testing / bug fixing

The project is a cooperation between Rajamuseo and XAMK (South-Eastern Finland University of Applied Sciences). It is scheduled to go live at Rajamuseo in 2023. Read more on the XAMK page.
